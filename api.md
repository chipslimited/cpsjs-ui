# 区块链浏览器的API

区块链浏览器的API分两类，一类直接调用jsonRPC，另一类需要通过调用封装的python web服务器间接调用。这两种类API的入口URL会有所区别

## 直接调用
由于安全性，javascript禁止跨站调用，因而需要在nginx里设立一个HTTP proxy， 代理服务器指向全节点的JsonRPC入口
直接调用的例子可以参见本目录下的swftc_example.js和eos.js
